Read Me
-----------

This installer includes Caarray2 build tag CAARRAY_R2_0_0_BETA6 from 12-20-2007 bundled with JBoss 4.0.5, MySQL 5.0.27, Firefox Portable Edition and JDK 1.5.0_10 for Windows.

The current installation assumes you're running on a clean Windows machine. The MySQL default password for root is blank. You may want to change this as you create a more robust system.

Before running the installer please close all other applications including any Firefox instances. Run install.bat and follow the wizards. You should be able to complete the installation by acception the defaults.

Known Issues
--------------
The installer will not close the command window after finishing. You can close the window manually after you click "Finish".
