1. Check out the released tag to SVN.
E.g.,  Check out CAARRAY_R2_1_0_ALPHA3 to F:\NCICB_CVS\CAARRAY_R2_1_0_ALPHA3

2. You will need to create on the fly a file "caarray-client.zip"

	a. > cd F:\NCICB_CVS\CAARRAY_R2_1_0_ALPHA3\software\build
	b. > ant build:caarray-client.zip
	
3. You now should have a file F:\NCICB_CVS\CAARRAY_R2_1_0_ALPHA3\software\caarray-client.jar\target\caarray-client.zip. For instance, for CAARRAY_R2_1_0_ALPHA3 it's 17,800 KB, and it has 359 files and folders. caarray-client.jar was 232 KB.

4. Unzip the F:\NCICB_CVS\CAARRAY_R2_1_0_ALPHA3\software\caarray-client.zip, e.g., to a newly created directory \software\caarray-client.jar\target\caarray-client_zip

5. There is documentation at E:\NCICB_CVS\CAARRAY_R2_0_2_PATCH1\software\caarray-client.jar\target\caarray-client_zip\README.txt

6. At the coommand prompt, run one or more of the following ant targets:

For local install, using the default ports of theBDA installer:
> E:
> cd F:\NCICB_CVS\CAARRAY_R2_1_0_ALPHA3\software\caarray-client.jar\target\caarray-client_zip
> ant -Dhostname=localhost -Djndi.port=31099 -Dgrid.port=18080

For QA-WM:
> E:
> cd F:\NCICB_CVS\CAARRAY_R2_1_0_ALPHA3\software\caarray-client.jar\target\caarray-client_zip
> ant -Dhostname=cbvapp-q1001.nci.nih.gov -Djndi.port=8080 -Dgrid.port=59580

For QA-Standalone:
> E:
> cd F:\NCICB_CVS\CAARRAY_R2_1_0_ALPHA3\software\caarray-client.jar\target\caarray-client_zip
> ant -Dhostname=cbapp-q1002.nci.nih.gov -Djndi.port=8080 -Dgrid.port=59580

For STAGE, run 
> E:
> cd E:\NCICB_CVS\CAARRAY_R2_0_2_PATCH1\software\caarray-client.jar\target\caarray-client_zip
> ant -Dhostname=cbapp-p1001.nci.nih.gov -Djndi.port=8080 -Dgrid.port=59580

For PROD, run 
> E:
> cd E:\NCICB_CVS\CAARRAY_R2_0_2_PATCH1\software\caarray-client.jar\target\caarray-client_zip
> ant -Dhostname=cbapp-s1001.nci.nih.gov -Djndi.port=8080 -Dgrid.port=59580
