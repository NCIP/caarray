To load this Affymetrix data into caArray:

1. Log in.
2. Go to Manage Array Designs.
3. If Test3 is not already imported, then import it as a new Affymetrix array design.
4. Create an experiment with:
   + Title = QA Affymetrix Experiment for API Testing
   + Assay Type = Gene Expression
   + Provider = Affymetrix
   + Array Design = Test3
   + Organism = Plasmodium falciparum (ncbitax)
5. Go to Manage Data.
6. Upload test3_data.zip.
7. Select all files and then import.
8. Lock the experiment from edits.
9. Go to the Permissions page for the experiment and give the Public "Read" access.
