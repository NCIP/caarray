To load this Genepix data into caArray:

1. Log in.
2. Go to Manage Array Designs.
3. If JoeDeRisi-fix.gal is not already imported, then import it as a new Genepix array design.
4. Create an experiment with:
   + Title = QA Genepix Experiment for API Testing
   + Assay Type = Gene Expression
   + Provider = Genepix
   + Array Design = JoeDeRisi-fix
   + Organism = Oryza sativa (ncbitax)
5. Go to Manage Data.
6. Upload genepix_data.zip.
7. Select all files and then import.
8. Lock the experiment from edits.
9. Go to the Permissions page for the experiment and give the Public "Read" access.
