Extracting Data Programmatically by API
Data in caArray can be extracted programmatically using an API. caArray has been set up to allow the following analysis services to use the data:
•
geWorkBench
•
caIntegrator
•
WebGenome
caArray also provides a Grid client which allows software engineers to acquire data from caArray.
For more information about using the API to extract data, see the caArray 2.0 Technical Guide which can be downloaded from this site: https://gforge.nci.nih.gov/frs/?group_id=305https://gforge.nci.nih.gov/frs/?group_id=305.
