Editing an Array Design
In caArray 2.0, any logged in user can edit an array design. An array design already in the system can be replaced by a new array design by following the edit steps.
After data has been imported into an experiment, the array design associated with that data cannot be removed from the experimentâ€™s list of array designs.
To edit an array design, follow these steps:
1.
On the row corresponding to the array design, click the Edit icon ().
OR
1.
Open the array design by clicking on its name, and click the Edit button () at the bottom of the details page.
2.
All required fields become editable; enter any edits.
3.
From this page, you can initiate uploading of a new array design file to replace the existing file.
4.
Save any edit by clicking the Save button.
