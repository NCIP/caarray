Grid availability
For those Institutions that register their caArray instance on caGRID, the public data is available to the integrated tools that use the caGRID service. For more information, see https://cabig.nci.nih.gov/workspaces/Architecture/caGrid.
The ability to browse and search from the caArray user interface features across the Grid is not available in caArray 2.0.
