Managing Protocols
In caArray, you can create and manage protocol(s) for referencing in an experiment. A protocol provides detailed documentation about the precise actions taken in any procedure that might be part of an experiment. For example, a protocol could describe the steps a laboratory used for any kind of process used in an experiment, such as the way a source material or sample is derived, the method used for labeling an extract or the methods used for running a hyrbridization or creating an image file of array results.
A protocol can be created independently of a specific experiment, or added during the course of creating an experiment. A protocol can be used by any caArray user but it can only modified by the owner of the protocol or another user with assigned permissions.
