Managing Array Designs
Any caArray user can upload, validate and import array designs for supported providers (Affymetrix, Illumina, and Genepix), and anyone can view, edit or replace the files. An array design only needs to be loaded once and is available to all users.
After data has been imported into an experiment, the array design associated with that data cannot be removed from the experimentâ€™s list of array designs.
You can also view array designs, or edit those for which you have permissions.
Manage Array Designs does not allow for the upload of array designs from providers for which caArray does not have parsers. If you attempt to upload such an array design, an intercept message will inform you of that fact.
