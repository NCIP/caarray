Files Types that can be Imported into caArray
The file types listed in the following table can be imported into caArray:

Acceptable File Formats

GenePix .GPR
•
Illumina .CSV

•
Affymetrix .cdf

Illumina Design .csv
•
Genepix .gal
•
Agilent .csv or .xml
•
UCSF Spot .spt
•
ImaGene .tpl
•
Nimblegen .ndf
Note: These can be uploaded, validated and imported only through the Manage Array Design feature described in Managing Array Designs.

MAGE-TAB ADF (Array Design Format)
Note: Any array design reference in a MAGE-TAB SDRF must refer to an array design that has already been imported into caArray.

MAGE-TAB with single SDRF (Sample and Data Relationship Format)
•
MAGE-TAB with multiple SDRFs

IDF (Investigation Description Format) only, no referenced SDRFs

SDRF with only source > Hybridizations

ADF (Array Design Format)

MAGE-TAB with existing samples in caArray.
