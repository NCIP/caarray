Browsing and Searching Before Login
Once you are at the caArray Portal Welcome login page, without being logged in you can browse caArray public data or you can perform a search of caArray public projects.
The Browse caArray dialog box on the left center of the page lists the number of experiments saved in the database in each of the listed categories. The text categories shown in blue hypertext format can be clicked to open additional pages that display experiments with attributes in that category.
The pages that open from the Browse dialog box list all caArray Experiments that have not been explicitly removed from visibility. For more information on the visibility options for Experiments, see Managing Experiment Visiblity.
•
For more information about browsing the caArray database, see Browsing the caArray Repository.
•
The Search caArray dialog on the right center of the page allows you to launch a search of the caArray database for public objects. For more information about executing a caArray search, see Searching the caArray Repository.
•
Options allowing you to submit, view, modify, and add microarray experiment data to caArray are dependent on your user privileges, once you are logged in.
You must obtain a user account in order to log in. For more information, see Requesting a User Account.
