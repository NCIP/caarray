Welcome to caArray 2.0
The following topics are top-level books in online help:
•
About caArray -- This section provides a caArray overview as well as a description of the relationship of caArray to caBIG™.
•
Getting Started in caArray --These topics provide details about launching caArray, navigating the interface, accessing online help and other links in the interface.
•
Navigating and Searching caArray --This section describes the processes for browsing the caArray repository and for conducting searches of the repository.
•
Creating and Managing Experiments --This section provides instructions for creating and managing Experiments in caArray.
•
Curation Tasks -- This section describes the processes for completing curation tasks in caArray relating to managing array designs, protocols and vocabulary terms.
•
Submitting Data to an Experiment --This section describes the processes for uploading, validating and importing annotation and array data into a caArray Experiment.
•
Extracting Data from caArray --This section describes the processes for downloading data from the caArray repository.
•
User Account Management --This section describes the process for creating user accounts and collaboration group accounts and managing the group accounts of caArray.
•
caArray References --This section includes descriptions and links to references relative to caArray technology and bioscience.
To learn more about each topic, browse through the online help, or use the table of contents, index or search features of help.
For more information, see Using caArray Online Help.
Context sensitive help is not implemented in caArray 2.0.
